KroTor-beta

Access the internet via Tor Network
To learn more about Tor Project,please visit https://www.torproject.org

  • On successful run ie 100% bootstrapped (see the logs in terminal), a tor (socks) proxy will be available system wide at 127.0.0.1:9999, point your browser/ application accordingly to the proxy
  • The app don't honor the system network settings yet, viz. if you are behind a proxy then it will not work for now
  • The functionality to define various settings such as firewall rules, proxy settings etc will be added in next version.
